
# K-Matrix Studio

Desktop editor for automotive bus description files.
Open, explore, edit, compare and merge DBC, ARXML, FIBEX and LDF files — in one tool.

## Supported formats

| Format | Protocols | Notes |
|--------|-----------|-------|
| `.dbc` | CAN / CAN FD | Multiplexing, attributes, value tables, comments |
| `.arxml` | CAN / FlexRay | AUTOSAR 4.x and 3.x |
| `.fibex` | CAN / FlexRay | ASAM FIBEX |
| `.ldf` | LIN | LIN 2.x |

Multiple files open simultaneously as tabs. Drag & drop supported.

---

## Features

- **Explore** — tree navigator, sortable/filterable tables for frames, signals, ECUs
- **Bit layout** — signal map with Motorola/Intel byte order, overlap detection, multiplexing
- **Communication matrix** — ECU × frames and ECU × ECU views
- **Edit** — all fields: network, bus, ECU, frame (E2E/SecOC blocks), signal, attributes
- **Create** — new database from scratch, top-down or bottom-up
- **Move signals** — relocate a signal to another frame
- **Merge** — merge two databases with per-frame conflict resolution
- **Compare** — frame-level diff between two open files
- **FlexRay** — schedule, matrix, and anatomy views
- **Export** — CSV and XLSX with column selection
- **Undo/Redo** — 50 steps per tab
